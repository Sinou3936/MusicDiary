# 음악 다이어리 기획안

> 오늘의 감정에 맞는 음악을 기록하는 감성 다이어리 서비스

---

## 서비스 개요

매일 듣는 음악에는 그날의 감정이 담겨 있다.
음악 다이어리는 감정 태그를 선택하면 어울리는 음악을 추천해주고, YouTube에서 찾은 음악을 일기처럼 기록할 수 있는 서비스다.

---

## 타겟 사용자

- 음악으로 감정을 표현하고 싶은 사람
- 나만의 음악 히스토리를 남기고 싶은 사람

---

## 핵심 기능

| 기능 | 설명 |
|------|------|
| 감정 태그 선택 | 행복, 우울, 설렘 등 감정 태그 1~3개 선택 |
| 음악 추천 | 선택한 감정 태그 기반으로 어울리는 음악 추천 |
| YouTube 검색 | 직접 곡을 검색해서 기록에 추가 |
| 메모 작성 | 그날의 감정을 짧은 글로 남기기 |
| 공개/비공개 | 기본 비공개, 원할 때 피드에 공유 가능 |
| 탐색 피드 | 다른 사람의 공개 음악 기록 탐색 |

---

## 사용자 플로우

```
로그인 (Google)
  → 감정 태그 선택
  → 음악 추천 또는 직접 검색
  → 곡 선택 + 메모 작성
  → 저장 (공개/비공개)
  → 내 타임라인에서 기록 확인
```
